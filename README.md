# 🎬 AI Video Watermark Removal

This project removes watermarks from videos using AI (IOPaint + OpenCV + FFmpeg).

---

## 🚀 Features

- 🔍 Automatic watermark detection (no manual ROI)
- 🎯 Object tracking using CSRT
- 🧠 AI inpainting using IOPaint
- 🎬 Frame-by-frame processing
- 🔊 Audio preserved from original video
- 📦 Batch processing support
- ⚡ CPU optimized pipeline

---

## 🛠 Tech Stack

- Python
- OpenCV
- FFmpeg
- IOPaint (AI model)
